# Documentação do Restaurante Flutter Firebase

Este projeto é um aplicativo de restaurante construído usando Flutter e Firebase. O objetivo principal é fornecer uma solução fácil para os usuários fazerem pedidos de refeições.

## Funcionalidades

- Menu interativo: Os usuários podem visualizar o menu do restaurante.
- Realização de pedidos: Os usuários podem fazer pedidos diretamente pelo aplicativo.
- Notificações em tempo real: Atualizações sobre o status do pedido são recebidas em tempo real.
- Gerenciamento do usuário: Permite que os usuários se registrem e façam login com suas credenciais.

## Tecnologias Utilizadas

- **Flutter**: Framework para construção de interfaces de usuário.
- **Firebase**: Backend como serviço que fornece autenticação, armazenamento em tempo real e muito mais.

## Como Configurar

1. Clone o repositório:
   ```bash
   git clone https://github.com/Jonatas1678/flutter-firebase-restaurant.git
   ```
2. Navegue até o diretório do projeto:
   ```bash
   cd flutter-firebase-restaurant
   ```
3. Instale as dependências:
   ```bash
   flutter pub get
   ```
4. Configure o Firebase:
   Adicione seu arquivo `google-services.json` para o diretório `android/app`.
